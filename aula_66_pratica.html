💻 Aula 66 — Bibliotecas de limpeza no contexto do projeto — prática de laboratório

📅 03/06/2026 ⏱ 50 minutos 🎯 Escola que Queremos 📚 Coleta e Limpeza de Dados I
Laboratório de informática

🎯 O que vamos aprender?

Nesta aula prática, a turma vai aplicar a lógica de limpeza diretamente a registros parecidos com os que chegam dos grupos do projeto. O objetivo é tratar a entrada dos dados da escola com mais organização, coerência e rastreabilidade.

💡 Aplicação no projeto: usar a lógica de limpeza como referência para tratar os registros que chegam dos grupos.

📘 Conteúdo principal

Conceito central: limpeza aplicada ao fluxo do projeto

Nesta prática, a turma vai simular o momento em que diferentes grupos entregam registros para a base comum do projeto. O trabalho será conferir a entrada, ajustar inconsistências e deixar os dados mais organizados para uso coletivo.

Roteiro da aula (50 min)

🧪 Atividade guiada de laboratório

Abra a planilha indicada pelo professor e siga este roteiro.

  1. Observe a origem dos registros: veja de qual grupo ou público cada linha veio.
  2. Compare formatos: procure diferenças na forma de registrar turma, data, local, resposta curta ou categoria.
  3. Ajuste inconsistências: padronize as informações iguais, corrija grafias divergentes e marque linhas duplicadas.
  4. Confirme a rastreabilidade: garanta que, mesmo após a limpeza, seja possível saber de onde veio cada dado.
  5. Finalize a base curta: salve a planilha organizada e anote quais regras de limpeza foram usadas.

📝 Resumo para o Caderno — Aula 66 Prática

  1. Os dados do projeto podem chegar de grupos diferentes e com formatos diferentes.
  2. A limpeza ajuda a unificar padrões e corrigir inconsistências.
  3. Padronizar não é mudar o dado, e sim deixá-lo coerente para uso coletivo.
  4. A origem do registro precisa continuar identificada depois da limpeza.
  5. Uma base organizada facilita a continuidade do projeto.
  6. A lógica de limpeza ajuda a transformar registros soltos em base utilizável.

✅ Fechamento da prática

Ao final da atividade, a turma deve perceber que a limpeza faz parte do caminho técnico do projeto e ajuda a transformar registros diferentes em uma base coerente e pronta para ser utilizada.

Ideia principal: aplicar a lógica de limpeza ao contexto do projeto ajuda a preparar uma base comum, organizada e rastreável.
